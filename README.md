# 月更志（MOGENGZHI） / mogengzhi-ios

**一句话定位**：把每个月当成一个版本来迭代与复盘（离线版）。

本仓库用于 iOS App「月更志」的实现。应用目标是以“月度版本迭代”的方式，帮助用户进行目标规划、待办推进与月末复盘，完全离线、无服务端依赖。

## 当前实现范围

- SwiftUI 应用骨架（4 个 Tab）
- 本地内存数据模型与演示数据
- 本月页：主题、目标、Backlog、Done 列表与状态流转
- Inbox 页：快速收集与归档
- 复盘页：复盘模板与数据概览
- 设置页：标签管理、备份与隐私说明占位

代码位于 `Mogengzhi/` 目录。
